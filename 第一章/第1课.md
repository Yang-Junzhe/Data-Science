## 数据类型
类型|记录方式
:---:|:---:
csv|用逗号去分隔开所有的数据
tsv|用Tab（制表符）去分隔开所有的数据
json|以键、值记录所有的数据

## 数据集的导入
* 导入numpy库:  
    ```python
    import numpy
    ```
* 导入数据集：  
    ```python
    arr = numpy.loadtxt('text_name.csv', str, delimiter = ',')
    # 这里的 arr 是一个数组
    ```
* 导入数据集时跳过第n行：
    ```python
    arr = numpy.loadtxt('text_name.csv', str, delimiter = ',', skiprows = n)
    ```

## 查看数据集的基本属性
语句|功能
