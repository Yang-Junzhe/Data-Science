## 数据集的求和、平均值、标准差和极值
numpy函数|功能
:---:|:---:
sum(arr)|求数据集arr的和
average(arr)|求数据集arr的平均值
std(arr)|求数据集arr的标准差
max(arr)|求数据集arr的最大值
min(arr)|求数据集arr的最小值
round(arr, n)|保留数据arr的n位小数

## 数据集的运算

### 数据集与常数之间的运算
```python
#使数据集arr中的每一个数据与n加减乘除
arr + n
arr - n
arr * n
arr / n
```
### 数据集与数据集之间的运算
```python
#使数据集arr_1与数据集arr_2中的对应数据加减乘除
arr_1 + arr_2
arr_1 - arr_2
arr_1 * arr_2
arr_1 / arr_2
```
