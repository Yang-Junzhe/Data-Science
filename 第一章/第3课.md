## 数据集的创建
1. `array`创建
    ```python
    arr = numpy.array([
        [1, 4, 5],
        [4, 7, 7],
        [3, 4, 5]
    ])
    ```
2. `zeros`创建
    ```python
    # zeros创建的数据集中的数据值均为0
    arr = numpy.zeros((3, 3))
    ```
3. `ones`创建
    ```python
    # ones创建的数据集中的数据值均为1
    arr = numpy.ones((3, 3))
    ```
4. `eye`创建
    ```python
    # eye创建的数据集形状为正方形，且每个所处横坐标与纵坐标相等的数据值为1，其余数据为值0
    arr = numpy.eye(5)
    ```
5. `arange`创建
    ```python
    # arange创建的数据集是一维的，且数据集中的数据的值为从1到100
    arr = numpy.arange(1, 101)
    
    # 此语句创建的数据集中数据的值为从1到100每增加2的所有数据
    arr_2 = numpy.arange(1, 101, 2)
    
    # resize可以改变数据集的形状，此语句将数据集arr的形状变为了10乘10的矩阵
    arr.resize((10, 10))
    ```
6. `randint`创建
    ```python
    # randint可以创建随机数据的数据集，此语句创建了一个形状为4乘4且数据的值为范围是从1到10的随机数的数据集
    arr = numpy.random.randint(1, 10, (4, 4))
    ```
    
## 数据集的计算

### 点乘
```python
arr_1 = numpy.array([
    [3, 4, 1],
    [2, 2, 5]
])
arr_2 = numpy.array([
    [1, 2, 4],
    [3, 6, 7],
    [2, 4, 5]
])
arr_3 = arr_1.dot(arr_2)
```
* 条件：  
    arr\_1的**行数**等于arr\_2的**列数**

* 算法：
    <details>
    <summary>查看</summary>

    1. 将arr\_1**第1行第1列**的值与arr\_2**第1行第1列**的值相乘，arr\_1**第1行第2列**的值与arr\_2**第2行第1列**的值相乘，arr\_1**第1行第3列**的值与arr\_2**第3行第1列**的值相乘，三次相乘的结果相加，得到arr\_3**第1行第1列**的值；  
    2. 将arr\_1**第1行第1列**的值与arr\_2**第1行第2列**的值相乘，arr\_1**第1行第2列**的值与arr\_2**第2行第2列**的值相乘，arr\_1**第1行第3列**的值与arr\_2**第3行第2列**的值相乘，三次相乘的结果相加，得到arr\_3**第1行第2列**的值；  
    3. 将arr\_1**第1行第1列**的值与arr\_2**第1行第3列**的值相乘，arr\_1**第1行第2列**的值与arr\_2**第2行第3列**的值相乘，arr\_1**第1行第3列**的值与arr\_2**第3行第3列**的值相乘，三次相乘的结果相加，得到arr\_3**第1行第3列**的值；  
    4. 将arr\_1**第2行第1列**的值与arr\_2**第1行第1列**的值相乘，arr\_1**第2行第2列**的值与arr\_2**第2行第1列**的值相乘，arr\_1**第2行第3列**的值与arr\_2**第3行第1列**的值相乘，三次相乘的结果相加，得到arr\_3**第2行第1列**的值；  
    5. 将arr\_1**第2行第1列**的值与arr\_2**第1行第2列**的值相乘，arr\_1**第2行第2列**的值与arr\_2**第2行第2列**的值相乘，arr\_1**第2行第3列**的值与arr\_2**第3行第2列**的值相乘，三次相乘的结果相加，得到arr\_3**第2行第2列**的值；  
    6. 将arr\_1**第2行第1列**的值与arr\_2**第1行第3列**的值相乘，arr\_1**第2行第2列**的值与arr\_2**第2行第3列**的值相乘，arr\_1**第2行第3列**的值与arr\_2**第3行第3列**的值相乘，三次相乘的结果相加，得到arr\_3**第2行第3列**的值
    
## 数据集的拼接

### hstack拼接
```python
# 将数据集arr_1和数据集arr_2左右拼接
numpy.hstack((arr_1, arr_2))
```
### vstack拼接
```python
# 将数据集arr_1和数据集arr_2上下拼接
numpy.vstack((arr_1, arr_2))
```

### concatenate拼接
```python
# 将数据集arr_1和数据集arr_2左右拼接
numpy.concatenate((arr_1, arr_2), axis = 1)

# 将数据集arr_1和数据集arr_2上下拼接
numpy.concatenate((arr_1, arr_2), axis = 0)
```
## 数据集的转置和求逆

### 转置
```python
#将数据集arr的所有数据的横坐标与纵坐标互换
arr.T
```
### 求逆
```python
#求矩阵arr的逆矩阵
numpy.linalg.inv(arr)
```
***
* *逆矩阵的定义*  
    设A是一个n阶矩阵，若存在另一个n阶矩阵B，使得
    ```math
    AB = BA =E
    ```
    则称方阵A可逆，并称方阵B为A的逆矩阵
